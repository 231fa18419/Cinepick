# 🎬 CinePick – Movie Recommendation Web Application

CinePick is a responsive movie recommendation web application developed to help users discover movies based on genres, ratings, popularity, and search preferences. The platform provides an interactive user experience with dynamic movie browsing, detailed movie information, and personalized recommendations using modern web technologies.

## 🚀 Features

* 🎥 Browse trending and popular movies
* 🔍 Search movies by title or keyword
* 📂 Filter movies based on genres and categories
* ⭐ View movie ratings, posters, and descriptions
* 📱 Responsive and user-friendly interface
* ⚡ Dynamic content rendering using JavaScript
* 🌐 API integration for real-time movie data

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* REST API
* JSON
* Git & GitHub

## 📌 Project Highlights

* Developed a responsive frontend interface for smooth user interaction across devices
* Implemented dynamic movie search and filtering functionality using JavaScript
* Integrated movie APIs to fetch and display real-time movie details
* Used asynchronous API calls and DOM manipulation for dynamic content updates
* Designed an interactive UI with modern styling and responsive layouts
